# Healthcare-management-system-SQL
Healthcare-management-system-SQL

Project Overview
The Healthcare Management System project is designed to analyze hospital data using MySQL.
It focuses on managing patients, doctors, appointments, treatments, and billing information to generate meaningful business insights for hospital management.
This project demonstrates real-world SQL data analytics skills commonly required in data analyst roles.

Project Objectives
Design a relational healthcare database using SQL
Perform data analysis using joins, aggregates, and subqueries
Analyze patient visits and doctor performance
Calculate hospital revenue and payment status
Generate insights to support healthcare decision-making
Tools & Technologies

Database: MySQL
IDE: MySQL Workbench
Language: SQL
Visualization: Microsoft Excel
Version Control: Git & GitHub

| Table Name   | Description                                      |
| ------------ | ------------------------------------------------ |
| patients     | Stores patient personal and registration details |
| doctors      | Stores doctor information and specialization     |
| appointments | Tracks patient-doctor appointments               |
| treatments   | Stores diagnosis and treatment cost              |
| bills        | Maintains billing and payment details            |
